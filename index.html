Présentation générales des TPs

UE 5.2 1er semestre:

HTML et CSS

Accédez ici aux travaux pratiques du 1er semestre dédiés à HTML et CSS : structure sémantique, mises en page (Grid, Flexbox) et exercices de style.

Vous trouverez le travail individuel, les exemples et le mini-projet ainsi que les ressources pour approfondir.

Javascript et Json

Accédez aux travaux pratiques JavaScript et JSON : manipulation du DOM, gestion d'événements, persistance (localStorage) et utilisation de données au format JSON.

Cette section regroupe les exercices en équipe, la logique applicative et le mini-projet To‑Do List.

UE 6.1 2ème semestre

PHP

Ressources et exercices autour de PHP : scripts serveur, traitement de formulaires et bases de données côté serveur.

Les TP montrent l'intégration backend côté serveur et les bonnes pratiques pour les formulaires sécurisés.

SQL et base de données

Travaux pratiques SQL et bases de données: modélisation, création de schémas, requêtes (SELECT, JOIN) et intégration avec PHP.

Comprend des exercices d'interrogation et des exemples d'intégration serveur-client.
