# README pour le correcteur — TPs de Laurent Boyer

Ce site statique présente l'ensemble des travaux pratiques (TP) réalisés par Laurent Boyer pour les UE 5.2 et 6.1.

Navigation rapide :

- Page d'accueil : `index.html` — liens vers les deux unités
- UE 5.2 (HTML / CSS / JS) → `S1-UE52-HTML-CSS-JS/home_S1.html`
- UE 6.1 (PHP / BD) → `S2-UE61-PHP-DB/home_S2.html`

Principaux dossiers :

- `S1-UE52-HTML-CSS-JS/` — TPs du 1er semestre (TPs HTML, CSS, JS, mini-projet ToDoList)
- `S2-UE61-PHP-DB/` — TPs du 2ème semestre (PHP / bases de données)

Si vous avez besoin d'une information précise sur un TP (fichiers sources, structure, ou explication), indiquez le numéro du TP et je fournirai les éléments demandés.

- `TPsHTML-CSS/` : TP1 → TP5 (pages HTML et exemples)
- `TP_JS/` : mini-projet ToDoList (index + Data/ JSON exemples + étapes)
- `Html/` et `Style/` : variantes et fichiers de style pour les exercices
- `S2-UE61-PHP-DB/` : pages et ressources pour le second semestre (PHP, bases de données). Le contenu sera complété au fur et à mesure des TPs.
